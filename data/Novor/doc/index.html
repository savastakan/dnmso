There are two version of Novor (1.1beta and 1.05) that I could have found. I tried both with default config files. The results were different. The latest version has higher matching rate.

1.1beta (2015.05.31)

The “1.1beta” version has been called as “v1.05.0573 (stable)” in output csv file.

1.05 (2015.11.10)

The “1.05” version has been called as “v1.06.0634 (stable)” in output csv file.

The version “1.05” brings “Cam” and “O” in parentheses with default config file. If these parentheses are ignored, the matching rate is increases. I added both results.
