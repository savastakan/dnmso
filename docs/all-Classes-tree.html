Class Hierarchy

 DNMSO; de novo ontology root class.
 Modification; PTM or general mass change.
 Either Terminal or ModifiedAminoAcid.
     ModifiedAminoAcid; An amino acid with PTM or other mass change.
     Terminal; terminal peptide modification.
 ModificationElement; container for possible Modification.
 Prediction; a de novo Prediction.
 Predictions; pontainer for multiple Prediction.
 Proof; either PeakValue or PeakLink.
     PeakLink; link to a peak in a spectrum.
     PeakValue; concrete peak value.
 Proofs; support for a SequenceElement.
 Publication; a journal or similar publication.
 Score; one of the Score for the Prediction.
 Scores; container for multiple Score.
 Sequence; the de novo predicted sequence.
 SequenceElement; an element of the predicted sequence (AminoAcid, ModAminoAcid, Gap).
     AminoAcid; child class of SequenceElement.
     Gap; an m/z range that couldn't be assigned an AminoAcid or a ModAminoAcid.
     Amino acid with PTM.
 Software; a de novo sequencing software.
 SoftwareSetting; one SoftwareSetting (parameter) with value.
 SoftwareSettings; container for multiple SoftwareSetting (parameters).
 Source; the data source for de novo prediction.
     Spectrum; a mass spectrum for de novo sequencing.
 Sources; the data sources for the de novo prediction.
 Spectra; a container for Spectrum.
 SpectrumData; CSV or SpecDataLink.
     SpectrumData; an MS/MS spectrum in CSV format.
     SpecDataLink; URL link to MS/MS spectrum.
