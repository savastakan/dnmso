Datatype Properties

aminoAcidName; full name of the amino acid (e.g., Glycine).
averageMass; average mass fo the modified amino acid.
averageModificationMass; The mass of the modifier (average isotope distribution).
combinedConfidence; the combined confidence for the prediction.
confidence; confidence for the SequenceElement.
csvPeakList; spectrum in CSV format.
cvParamName; The name of the parameter from the CV.
cvTermAccession; The accession of the parameter from the CV.
cvVersion; The version of the CV used.
fullName; The full name of the CV.
gapValue; m/z range without sequence information.
id; identifier of the controlled vocabulary.
isTerminalModification; whether the modification is terminal.
largerIsBetter; which score value is better?
mainScore; default score.
modAAid; unique identifier for ModifiedAminoAcid.
modificationName; name of the modification.
monoIsotopicMass; monoisotopic mass of amino acid. Either monoIsotopicMass or averageModificationMass should be provided.
monoModificationMass; The mass of the modifier (monoisotopic).
peakID; The ID of the peak in the Spectrum.
peakId; the intensity of a peak.
peakMZ; the m/z of a peak.
peptideSequence; sequence of the peptide.
positionInSequence; position within Sequence.
precursorIntensity; ihe intensity of the precursor peak.
precursorMZ; the m/z of the precursor peak.
PSI-Mod-Ref; The identifier provided by PSI-Mod.
pubUrl; the link to a publication.
scanId; the scan number of the spectrum.
scoreName; name of the Score.
scoreValue; value for the score.
softwareName; name of the de novo sequencing tool.
softwareSettingName; name of the SoftwareSetting (parameter).
softwareSettingValue; softwareSetting (parameter) value.
softwareVersion; version of the de novo sequencing tool.
spectrumId; the spectrum ID (unique identifier per DNMSO instance).
threeLetterCode; an amino acids abbreviation.
unitAccession; The accession to the unit used.
unitCVRef; The CV the unit is described.
unitName; The name of the unit.
uri; URI pointing to the CV.
url; link to spectra.
usrValue; the value of the CV term used.
version; the version of DNMSO used for the instance.
