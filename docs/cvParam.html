Class 'cvParam; A parameter described via a controlled vocabulary.'

type
Class [owl:Class]
comment
cvParam allows the specification of information using a CV. The CV used must be specified in DNMSO and linked in cvParam via the foundIn association. Some parameters are required (cvParamName, cvTermAccession, usrValue) and some are optional (unitAccession, unitCVRef, unitName).
