Class 'CV; A controlled vocabulary used in the instance of DNMSO.'

Class [owl:Class]
comment
Many fields in DNMSO are not using a controlled vocabulary as the de novo sequencing standardization field has to develop first. However, the intention is to include CVs and the CV class allows the specification of the CVs used in an instance. The future aim is to use PSI-MS and incorporate de novo terms in there.
