comment
This is a container for Prediction. It is possible that multiple predictions can be made for one underlying spectrum. For example co-fragmentation would lead to several valid sequences. Alternatively, multiple predictions can be made and represented in DNMSO with varying sequences and confidence and scores. Predictions merely groups the Prediction made for one or more mass spectra. A Prediction is not required for the DNMSO instance to be valid.
