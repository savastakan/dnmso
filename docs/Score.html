comment
De novo sequencing tools often provide multiple scores, this represents one of them. The scoreName, the actual scoreValue for the Prediction has to be provided. For the Score it needs to be set whether a larger Value is better or not (largerIsBetter). In the future, the intention is to use a controlled vocabulary so the use of cvParam (hasScorecvParam) is encouraged.
