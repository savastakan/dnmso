Property 'gapValue; m/z range without sequence information. '

type
DatatypeProperty
comment
The m/z value for the Gap which couldn't be filled with an AminoAcid or a ModAminoAcid. The intended mass is monoisotopic. The intention is to avoid filling in arbitrary sequences with similar masses but clearly state that the range is ambigous for the software, settings, and spectra used. Perhaps a differnent software or an additional spectrum could resolve the issue.
